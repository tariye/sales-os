Working Notes

What this shell shows

Only active data-plane state, read-only legacy archive inventory, and pilot-labeled stock analysis.

What is hidden

New Entry, Queue, Command Center, legacy Live Signals, Asset Lab, Memory DB, Actions, Patterns, Changelog, and sample-data controls are not in the primary navigation.

Where to look next

Use Sources for state, Evidence for immutable snapshots, System Health for scheduler and worker truth, and Build Information for version and schema details.
